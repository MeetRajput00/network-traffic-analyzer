# Network Traffic Analyzer

This repository contains a network traffic analyzer built using SharpCap library. It is used to filter out any harmful traffic in your network device.

## Installation

You first need to download the Npcap library online. You can find it in npcap.com . This library allows the build to read the traffic in selected network device.

To get started, you can clone this repository to your local machine using the following command:

```bash
git clone https://github.com/MeetRajput00/network-traffic-analyzer.git
```
